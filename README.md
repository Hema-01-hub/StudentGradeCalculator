# Student Grade Calculator - Java

## Description

A Java application that calculates total marks, average percentage, and grade based on student scores.

## Features

* Marks Input
* Total Marks Calculation
* Percentage Calculation
* Grade Generation
* Result Display

## Technologies Used

* Java
* OOP Concepts
* Conditional Statements
* Loops
* Scanner Class

## How to Run

```bash
javac StudentGradeCalculator.java
java StudentGradeCalculator
```
